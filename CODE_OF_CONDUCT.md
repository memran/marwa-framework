# Code of Conduct

## Our Standard

This project follows a professional, respectful, and inclusive collaboration standard. Contributors are expected to communicate constructively, review code in good faith, and focus discussions on improving the project.

Examples of expected behavior:

- Giving and receiving feedback respectfully
- Assuming good intent while discussing technical disagreements
- Using clear, professional language in issues, pull requests, and reviews
- Respecting different experience levels and backgrounds

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks
- Discriminatory or exclusionary language
- Public or private intimidation
- Repeated off-topic disruption in project discussions

## Scope

This Code of Conduct applies to all repository spaces, including issues, pull requests, discussions, review comments, and other project-related communication channels.

## Enforcement

Project maintainers may remove comments, reject contributions, or restrict participation for behavior that violates this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainer at `memran.dhk@gmail.com`.
